Django2.0学习记录

基本使用

pip install Django

创建一个新的项目和应用

Django-admin startproject cms // 项目
cd cms
python manage.py startapp we //应用

创建数据库并同步

python manage.py makemigrations web //创建数据库
python manage.py migrate //同步数据库

创建超级用户

python manage.py createsuperuser

交互式shell

python manage.py shell

on_delete(表关系)

on_delete=models.CASCADE级联删除
当删除主表的数据时候从表中的数据也随着一起删除

auto_now_add(DateTime)

auto_now_add为添加时的时间，更新对象时不会有变动

objects.all()

all返回的是QuerySet对象，程序并没有真的在数据库中执行SQL语句查询数据，但支持迭代，使用for循环可以获取数据

objects.get()与objects.filter()二者类似

get返回的是Model对象，类型为列表，说明使用get方法会直接执行sql语句获取数据

用于类型转换，将object的类型转换为可读的str型
def str(self):
return self.name

urls模块

int:question_id

path(‘int:question_id/‘, views.detail, name=’detail’)
<int:是一个转换器，用于确定哪些模式应与URL路径的这一部分匹配
:question_id>定义将用于标识匹配模式的名称

view模块

markdown渲染

pip install markdwon

import markdown

def detail(request, pk):
post = get_object_or_404(Post, pk=pk)
post.body = markdown.markdown(post.body,
extensions=[
‘markdown.extensions.extra’,
‘markdown.extensions.codehilite’,
‘markdown.extensions.toc’,
])
return render(request, ‘blog/detail.html’, context={‘post’: post})
